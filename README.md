# AnvilCores

A collection of Anvil Series RISC-V CPU cores written in SystemVerilog.

## Cores

| Core | Description | Status |
|------|-------------|--------|
| [Anvil-Ultra](https://github.com/JohnH2448/Anvil-Ultra) | RISC-V Out-of-Order M-Mode RV32I Zicsr Core in SystemVerilog | Coming soon |
| [Anvil-Pro](https://github.com/JohnH2448/Anvil-Pro) | RISC-V Dual-Issue 6-Stage RV32I Zicsr Reorder-Buffered Core in SystemVerilog | In progress |
| [Anvil-Core](https://github.com/JohnH2448/Anvil-Core) | RISC-V Dual-Issue 6-Stage RV32I Zicsr Core in SystemVerilog | In progress |
| [Anvil-Lite](https://github.com/JohnH2448/Anvil-Lite) | RISC-V Lightweight M-Mode RV32EC Zicsr Core in SystemVerilog | Coming soon |
| [Anvil-Demo](https://github.com/JohnH2448/Anvil-Demo) | RISC-V 5-Stage RV32I Zicsr Core in SystemVerilog | In progress |

## Notes

This repository serves as an index for the Anvil core family. Each core is maintained in its own standalone repository.
